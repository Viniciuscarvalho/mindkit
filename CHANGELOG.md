# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-01-20

### Changed

- Improved tool detection to check for CLI commands in PATH (claude, cursor, codex) in addition to config directories

## [1.0.0] - 2026-01-19

### Added

- Initial release of mindkit
- Support for Claude Code, Cursor, and Codex CLI tools
- Commands: install, backup, list, init, sync
- Template system with transforms for cross-tool compatibility
